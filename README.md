# 海外小游戏广告测评助手 (GameAd Insight Pro)

一款专为游戏测试员设计的 React 应用 (PWA)，用于记录海外小游戏试玩数据、分类游戏类型、追踪广告策略，并支持一键导出 Excel。

## ✨ 主要功能

*   **详细数据录入**：记录游戏基本信息、试玩时长、广告策略组（支持拖拽排序）。
*   **灵活的广告模块**：
    *   支持添加多个广告策略组。
    *   内置“游戏时间/进度”标记。
    *   自定义广告属性模板（如广告类型、时长、触发条件等）。
*   **本地离线运行 (PWA)**：
    *   支持安装到桌面或手机（Android/Windows/macOS）。
    *   基于 Service Worker 的离线缓存，无网环境也能完整使用。
    *   数据存储在本地浏览器 (LocalStorage)，安全且私密。
*   **数据导出**：一键导出为格式化的 Excel (.xlsx) 表格。
*   **移动端适配**：响应式设计，针对手机操作优化的 UI。

## 🚀 如何运行

本项目采用现代 **ES Modules** 和 **Import Maps** 技术，无需复杂的 Node.js 构建流程（如 Webpack/Vite），直接在浏览器中运行。

1.  **克隆或下载**本项目代码到本地。
2.  **启动本地静态服务器**。由于使用了 ES Modules，直接双击 `index.html` 可能无法运行（会有 CORS 跨域限制），你需要一个静态服务器。
    *   **VS Code (推荐)**: 安装 "Live Server" 插件，右键 `index.html` 选择 "Open with Live Server"。
    *   **Python**: 在项目根目录下打开终端，运行 `python -m http.server`，然后访问 `http://localhost:8000`。
    *   **Node.js**: 运行 `npx serve .`。

## 📁 项目结构
